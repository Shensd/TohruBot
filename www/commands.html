Commands

The most important feature of any bot: commands!
Tohru bot features many commands, and they have all been listed here for your reading pleasure!

List of Commands

Name Parameters Description Example Admin
echo [Content to be repeated] Echos back the given input $echo hello world! No
ping None Pong! $ping No
thonk None Hmmmm $thonk No
play [youtube url or search query] Plays a given song in the user's voice channel $play https://www.youtube.com/watch?v=dQw4w9WgXcQ No
skip None Skips the currently playing song $skip Yes
queue None Lists the currently queued songs $queue No
clear None Clears the queue of songs $clear Yes
link None Gives a link and description of currently playing song $link No
